# FF14 로그 🪶

**파이널판타지14(FFXIV) 채팅 로그를 캐릭터별로 예쁘게 정리해주는 웹 툴**

게임에서 복사한 채팅 로그를 붙여넣으면, 닉네임마다 색상·프로필을 입혀 보기 좋은 대화 형태로 변환해줍니다.
카페·블로그·디스코드에 공유하거나 이미지로 저장하기 좋아요.

👉 **[바로 사용하기](https://kjn4101.github.io/ffxiv-chatlog/)**

---

## ✨ 주요 기능

### 캐릭터 설정
- 닉네임마다 **표시 이름 · 배경색 · 글씨색 · 프로필**(이모지 / 사진) 지정
- 사진은 **드래그 + 확대**로 원하는 부분을 잘라 등록
- 색상은 팔레트로 고르거나 **색상 코드(#RRGGBB)** 직접 입력
- **검색**으로 캐릭터를 빠르게 찾고, 로그에 등장하는 캐릭터만 자동으로 추려보기
- 설정은 브라우저에 저장되어 다음에도 그대로 사용

### 로그 변환
- 로그를 붙여넣으면 **자동 변환**
- 닉네임 앞 장식 문자, 닉네임 뒤 **서버명**(초코보/모그리/펜리르/카벙클/톤베리) 자동 정리
- **채널 필터**로 보고 싶은 채널만 표시 (전체 선택 지원)
- **시간 표시 · 채널 이름 표시** 켜고 끄기
- **로그 배경색 · 시스템 글자색** 변경

### 말풍선 종류별 표현
- **일반 대화** — 캐릭터 색 말풍선
- **귓속말** — 보낸/받은 모두 `보낸사람 → 받은사람`, 조용한 톤으로 구분
- **감정표현** — 나래이션처럼 가운데 정렬
- **시스템 알림** — 공지·토벌 종료 등은 한 줄로 차분하게

### 내보내기
| 방법 | 어울리는 곳 |
|------|------------|
| **서식 복사하기** | 구글 문서 · 워드 · 카페 · 메일 |
| **HTML 코드 복사** | 티스토리 · 블로그의 'HTML 편집' 모드 (미리보기 모습 그대로) |
| **텍스트만 복사** | 디스코드 · 메모장 |
| **이미지로 저장** | 전체 / 보이는 부분 PNG 캡처 |

## 📋 인식하는 로그 형식

| 형식 | 예시 |
|------|------|
| `[채널]<닉네임> 메시지` | 자유부대 · 링크셸 · 서초링 등 |
| `(닉네임) 메시지` | 파티 |
| `닉네임: 메시지` | 일반 채팅 |
| `>>닉네임: 메시지` / `닉네임 >> 메시지` | 귓속말 (보냄 / 받음) |
| `[이벤트] 메시지` | 시스템 공지 |
| `닉네임이/가 ~합니다` | 감정표현 (닉네임 등록 필요) |

> 인식하지 못하는 형식이 있으면 알려주세요.

## 🚀 사용법

1. **[페이지 열기](https://kjn4101.github.io/ffxiv-chatlog/)**
2. 왼쪽에서 **캐릭터 추가** → 닉네임·색·프로필 설정
3. 게임에서 채팅 로그를 복사해 **로그 입력칸에 붙여넣기**
4. 미리보기에서 확인 → **복사 / 이미지 저장**

---

## 🔒 개인정보

모든 처리는 **브라우저 안에서만** 이뤄집니다. 입력한 로그나 캐릭터 설정은 서버로 전송되지 않고, 캐릭터 설정만 사용 중인 브라우저에 저장됩니다.
